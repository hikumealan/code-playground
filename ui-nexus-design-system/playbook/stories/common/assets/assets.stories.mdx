Sketch Design Kit
=================

Designers can easily use the Nexus Design System by following these steps:

1. Install Sketch - you must have [Sketch](https://www.sketch.com/) installed to use our Nexus Design Library
2. Get Nexus Design Library - download the recent version of the [Design
Assets](https://sketch.cloud/s/6cacde0d-b875-4d50-8d2a-e938f09fc7a9)
3. Select "Add/Update Library to Sketch"

* Install Plugin - you will need the [Camilo](https://github.com/Pausansol/Camilo) plugin for managing the Library
versions - edit the library version of your sketch file by: going to `plugins` and running the `"Camilo" plugin` >
select the `nexus library` and select `swap`
